## ✨ Features

| Feature | Description |
|---|---|
| 📋 **5-Step Guided Form** | Personal Info → Summary → Experience → Education → Skills |
| 👁️ **Live Preview** | Your resume updates in real-time as you type |
| 🎨 **3 Professional Templates** | Modern Purple, Minimal Clean, Executive Dark |
| 📄 **PDF Download** | One-click print-to-PDF — works in any browser |
| 💾 **Auto Save** | Your progress is saved automatically to browser storage |
| ➕ **Repeatable Sections** | Add multiple jobs, degrees, and skills with ease |
| ⚡ **Suggested Skills** | Quick-add common skills with one click |
| 🔗 **Deep Link Templates** | Share direct links to specific templates via URL params |

## 🚀 Getting Started

### Option 1: Open Directly
Just double-click `index.html` — it runs in any browser, no server needed.

### Option 2: Local Server (Recommended)
```bash
# Clone the repo
git clone https://github.com/muhammadusman2228/resume-builder.git

# Serve locally
npx serve . -p 3000

# Open in browser
# http://localhost:3000/resume-builder/index.html
```

## 💡 How to Use

```
1. Open the app → Click "Build Resume"
2. Fill in Step 1: Personal Info (Name, Email, Title, Location)
3. Fill in Step 2: Write a 2-3 sentence professional summary
4. Fill in Step 3: Add your work experience or internships
5. Fill in Step 4: Add your education / degrees
6. Fill in Step 5: Add your skills (or click quick-add suggestions)
7. Switch templates using the toolbar (Modern / Minimal / Executive)
8. Click "Download PDF" → Save as PDF from the print dialog
```
